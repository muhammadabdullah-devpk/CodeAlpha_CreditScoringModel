Developer Integration

CreditAI REST API

Integrate real-time credit risk analytics, loan probability metrics, and explainability reasoning directly into your existing banking CRM or digital ledger.

Authentication

The CreditAI API currently permits public sandbox queries. Production keys should be passed as a standard bearer token in the headers array for rate-limit allocations.

Authorization: Bearer YOUR_API_KEY
POST

/api/predict

Send individual applicant details to evaluate creditworthiness, approval probability, and detailed risk factors.

Request Body Properties (JSON)
Field Type Example Description
age integer 35 Age between 18 and 80.
annual_income number 75000 Gross annual earnings.
monthly_debt number 1500 Cumulative monthly debt.
credit_utilization number 0.30 Credit card utilization (0.00 to 1.00).
home_ownership string "mortgage" "own", "mortgage", or "rent".
loan_purpose string "personal" "business", "personal", "car", "home_improvement".
Example Request
curl -X POST http://127.0.0.1:5000/api/predict \
  -H "Content-Type: application/json" \
  -d '{
    "age": 30,
    "annual_income": 68000,
    "monthly_debt": 1200,
    "employment_length_years": 4,
    "credit_history_years": 6,
    "loan_amount": 15000,
    "loan_term_months": 36,
    "open_accounts": 5,
    "credit_utilization": 0.28,
    "late_payments_12m": 0,
    "delinquencies": 0,
    "bankruptcies": 0,
    "savings_balance": 18000,
    "previous_defaults": 0,
    "home_ownership": "rent",
    "education_level": "bachelor",
    "employment_type": "full_time",
    "loan_purpose": "personal"
  }'
import requests

    "age": 30,
    "annual_income": 68000,
    "monthly_debt": 1200,
    "employment_length_years": 4,
    "credit_history_years": 6,
    "loan_amount": 15000,
    "loan_term_months": 36,
    "open_accounts": 5,
    "credit_utilization": 0.28,
    "late_payments_12m": 0,
    "delinquencies": 0,
    "bankruptcies": 0,
    "savings_balance": 18000,
    "previous_defaults": 0,
    "home_ownership": "rent",
    "education_level": "bachelor",
    "employment_type": "full_time",
    "loan_purpose": "personal"
}

response = requests.post(url, json=payload)
print(response.json())

/api/metrics

Response Example

  "precision": 0.7184,

/api/feature-importance
